# Screenshots для PWA

Эта папка содержит скриншоты приложения для отображения в App Install промпте.

## Требования к скриншотам

### Desktop (Wide Form Factor)
- **Размер**: 1920x1080px (16:9)
- **Формат**: PNG или JPEG
- **Максимальный размер файла**: 8MB
- **Минимум**: 1 скриншот
- **Рекомендуется**: 3-5 скриншотов

### Mobile (Narrow Form Factor)
- **Размер**: 750x1334px (iPhone 8)
- **Формат**: PNG или JPEG
- **Максимальный размер файла**: 8MB
- **Минимум**: 1 скриншот
- **Рекомендуется**: 3-5 скриншотов

## Необходимые скриншоты

Создайте следующие скриншоты приложения:

1. **desktop-letters.png** - Список писем на десктопе
2. **mobile-letters.png** - Список писем на мобильном
3. **desktop-detail.png** - Детали письма на десктопе
4. **mobile-detail.png** - Детали письма на мобильном
5. **desktop-settings.png** - Настройки на десктопе
6. **mobile-settings.png** - Настройки на мобильном

## Как создать скриншоты

### Desktop
1. Откройте приложение в браузере в режиме desktop
2. Установите разрешение окна 1920x1080
3. Используйте DevTools (F12) → Screenshot или расширение для скриншотов
4. Сохраните в формате PNG

### Mobile
1. Откройте DevTools (F12)
2. Включите Device Toolbar (Ctrl+Shift+M)
3. Выберите устройство "iPhone 8" или установите размер 750x1334
4. Сделайте скриншот устройства
5. Сохраните в формате PNG

## Best Practices

- Используйте реальные данные приложения
- Показывайте ключевые функции
- Избегайте пустых экранов
- Убедитесь, что UI выглядит профессионально
- Не включайте личные данные пользователей
- Используйте темную тему приложения для консистентности
